# Breezy-The_SocialMedia_WebApp
>social media website for everyone
## To use
* Install Dependencies using
>using npm
```npm
  npm i && cd client npm i
```
>using yarn
```yarn
  yarn install && cd client yarn install
```
* To Start the client && server side app
>using npm
```npm
  npm start
```
>using yarn
```yarn
  yarn start
```
